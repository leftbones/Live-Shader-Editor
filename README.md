# Live Shader Editor

A real-time shader editor for creating and testing GLSL shaders with instant visual feedback!

## Features

- Real-time shader compilation and rendering
- Syntax highlighting for GLSL code
- Support for saving and loading external .gdshader files
- Shader parameter controls for the shader preview

## Roadmap

- CanvasItem shader support (currently only supports Spatial shaders)
- Web-based version (needs a custom solution for file system access)
- Multiple editor tabs
- Editable keyboard shortcuts
- Improved shader error reporting
